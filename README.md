# 2. 프로젝트 개요
---

## 📌 주제
- 문서 요약 웹 퍼블리케이션을 활용하여 PDF, TXT 등의 문서를 요약하는 서비스

## 💡 주제를 선택한 이유
- 다양한 문서 형식을 효과적으로 요약하여 정보 접근성을 높이기 위해

## 🚀 주요 기능
1. **회원 관리**
   - 회원가입 및 로그인 기능
2. **문서 요약**
   - 파일 업로드 시 문서 요약 기능 제공
